Logan

Parse and analyze IIS and S3 access logs with SQL. Fast, cross-platform, Parquet-powered.

curl -fsSL https://wagneripjr.github.io/logan-pub/install.sh | sh

Features

Format Auto-Detection

Automatically identifies IIS W3C Extended and AWS S3 Server Access Log formats. Just point it at a file.

SQL Queries

Run arbitrary SQL against your logs using DataFusion. Filter, aggregate, and join with full SQL syntax.

Parquet Caching

Parsed logs are cached as Parquet files. Subsequent queries skip parsing entirely for instant results.

S3 Direct Access

Analyze S3 access logs directly from a bucket with s3:// URIs. ETag-based caching avoids re-downloads.

Built-in Reports

Pre-built commands for top endpoints, slowest requests, error rates, client analysis, and time-series breakdowns.

Cross-Platform

Native binaries for Linux (amd64, arm64), macOS (Apple Silicon), and Windows. No runtime dependencies.

Quick Start

# Parse and show log summary
logan -s access.log info

# Top 20 endpoints from a directory (recursive)
logan -s /var/log/iis/ -r top 20

# Analyze S3 access logs directly from a bucket
logan -s "s3://my-bucket/logs/" top 10

# Run custom SQL
logan -s access.log sql "SELECT client_ip, COUNT(*) as n FROM logs GROUP BY 1 ORDER BY 2 DESC LIMIT 10"

# Show errors with local timezone
logan -s access.log --local errors
